# Excavator

A Minecraft Paper 1.21.1 plugin.

## Features

- Paper 1.21.1 support
- Java 21
- Gradle build
- Example plugin structure

## Building

Build with:

```sh
./gradlew build
```

JARs are output to `build/libs/`.
